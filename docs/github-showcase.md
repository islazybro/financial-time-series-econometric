# Como presentar este proyecto en GitHub

## Nombre sugerido del repositorio

- `financial-time-series-econometrics`
- `econometric-analysis-bbva-santander`

## Descripcion corta del repositorio

Proyecto de econometria financiera en Python para analizar estacionariedad, volatilidad e interdependencia dinamica entre las acciones de BBVA y Santander.
