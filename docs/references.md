# Referencias conceptuales

Este proyecto se inspira en el enfoque metodologico de dos textos de apoyo:

## Stock y Watson

**Introduccion a la Econometria**

La idea mas valiosa para este repositorio es que la teoria debe acompañar a la aplicacion. Por eso la documentacion no se limita a mostrar codigo o resultados, sino que explica la pregunta economica detras de cada paso.

## Enders

**Applied Econometric Time Series**

Este texto sirve como referencia natural para el tratamiento de series de tiempo, especialmente en temas como:

- estacionariedad,
- modelos ARIMA,
- volatilidad condicional,
- sistemas VAR,
- causalidad e impulso-respuesta.

## Como se usan en el proyecto

- Stock y Watson orienta la forma de explicar.
- Enders orienta la estructura de analisis de series de tiempo.
- El proyecto original en R aporta el caso aplicado sobre BBVA y Santander.
